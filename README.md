# Brain-Games
The project contains several games, where you see diffrent questions about numbers and your goal is to answer correctly.

## Brain-Even
Answer "yes" if the number is even, otherwise answer "no"
[![brain-even](https://asciinema.org/a/AG01RP9JN3qAd6Kc)](https://asciinema.org/a/AG01RP9JN3qAd6Kc)

## Brain-Calc
What is the result of the expression?
[![brain-calc](https://asciinema.org/a/DsnkR7xtdvCFrTUv)](https://asciinema.org/a/DsnkR7xtdvCFrTUv)

## Brain-GCD
Find the greatest common divisor of given numbers.
[![brain-gcd](https://asciinema.org/a/YyhDeF2by1fAhobov)](https://asciinema.org/a/YyhDeF2by1fAhobo)

## Brain-Progression
What number is missing in the progression?
[![brain-progression](https://asciinema.org/a/XnvH3ZQ4SHTCxZja)](https://asciinema.org/a/XnvH3ZQ4SHTCxZja)

## Brain-Prime
Answer "yes" if given number is prime. Otherwise answer "no".
[![brain-prime](https://asciinema.org/a/psZveCwxjD3kkzPs)](https://asciinema.org/a/psZveCwxjD3kkzPs)


# Installation:

Package is built by running the command: 
```make build```
The installation can be done by running the command: 
```package-install```

After that games can be run by game names:
```brain-games```
```brain-even```
```brain-calc```
```brain-gcd```
```brain-progression```
```brain-prime```
